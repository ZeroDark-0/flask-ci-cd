# Flask CI/CD with Jenkins and Docker

This repository demonstrates a simple CI/CD pipeline that builds and deploys a Dockerized Flask application using Jenkins.

- 🐍 Python 3.9
- ⚗️ Flask
- 🐳 Docker
- ⚙️ Jenkins
- ☁️ AWS EC2 (Ubuntu host)
- 🐙 GitHub
